# Video Platform Front End

This is a video platform app to upload videos using imagekit and Django

## Tech Used
- React
- React Router

## How to run it

The project is setup to run using nom to make it easier to run for everybody.

First check if your machine has NodeJS and npm installed by opening a console and running this command:

``` BASH
npm --version
```

If there is an error just follow the [NodeJS documentation](https://nodejs.org/en/download) to install NodeJS and npm in your machine


When you have npm installed just clone the repository, open a console inside the root of the project and run:

``` BASH
npm install
npm run dev
```
